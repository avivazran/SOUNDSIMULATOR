WHAT IS SOUNDSIMULATOR?


SOUNDSIMULATOR is a physically-accurate room acoustics sandbox. It allows the user to place sound sources and microphones in specific locations, define their orientation, sound wave and movement parameters and experience the sound as it is heard through each of the microphones with respect to the material the room is made of, as well as its dimensions. Utilizing STEAM AUDIO as its audio plugin and Intel Embree for ray-tracing kernels, sound-related phenomena such as attenuation, occlusion, reflection, absorption, doppler and more are detectable. The simulator enables the user to run a simulation which will produce .wav files containing the sound as it was heard in each microphone. furher more, it comes with a python API for running automated simulations.
